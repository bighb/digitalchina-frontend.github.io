# Buffet

> 开发者友好，智能，快速，可扩展的 管理后台 设计器 与 脚手架 构建系统

> 低代码系统

## Features 功能

- 一个可以部分代替 `既有原型设计` 工具的 设计器
  - 所见即所得的画布
  - 反向生成 `既有原型设计` 工具的设计文件
- 一个可以 脚手架 代码一键生成的 构建系统

  - 脚手架生成即可使用

    - 支持多种技术栈

      - Vue3
      - Vue
      - React
      - JQ (IE8 Compatible)

    - 支持定制化

      - 较低的 `二开` 学习成本

        > 和学习一个脚手架的成本相当

  - 脚手架更新
    - 自动更新
    - 手动更新
